В эту директорию можно добавить любые файлы, которые потребуются приложению во время выполнения.

Этот вспомогательный файл можно удалить в одном из двух случаев:
1) в этой директории есть хотя бы один другой файл;
2) в этой директории нет файлов (или даже нет самой директории), а в CMakeLists.txt убрана секция, которая отвечает за копирование этой директории.
